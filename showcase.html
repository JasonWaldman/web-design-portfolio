Web Design Portfolio Project

This project is a multi-page portfolio website designed and developed as part of MFJS 3501. The goal was to create a clean, accessible, and responsive site that presents my work and professional background while demonstrating core web design and development skills.

Project Overview

The site was built to function as a professional portfolio that communicates my experience in communication strategy and web design. The design focuses on clear navigation, readable layout, and strong visual hierarchy so users can quickly understand who I am and what work I have created. Throughout the project I applied mobile-first design principles, improved accessibility, and refined the layout through multiple iterations based on usability feedback.

Project Screenshots

Homepage

The homepage introduces the portfolio, highlights my professional focus, and provides clear navigation to the major sections of the site including About, Portfolio, and Contact.

Interior Page

Interior pages expand on the content and demonstrate consistent structure, accessible headings, readable spacing, and clear hierarchy that helps users scan information quickly.

Portfolio Section

The portfolio section highlights selected work using reusable card components that present project descriptions, images, and role explanations in a clear and organized layout.

What I Learned

Through this project I gained hands-on experience building a complete website from scratch using HTML, CSS, and JavaScript. I strengthened my understanding of responsive design, accessibility best practices, and how layout and hierarchy influence usability. I also became more comfortable using GitHub for version control and publishing a live website through GitHub Pages. The project helped me see how thoughtful design decisions and small usability improvements can significantly improve the overall user experience of a website.
